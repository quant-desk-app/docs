## Quick answers

### Is QuantDesk custodial?

No. Wallet keys remain in user wallets.

### Should I start on devnet?

Yes. Use devnet for first-time setup and workflow testing.

### Where is endpoint schema truth?

Swagger/OpenAPI remains the source of truth for endpoint-level payload details — see [Developer quickstart](../developers/quickstart).

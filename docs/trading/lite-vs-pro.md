# Lite vs Pro terminal

Use the mode that matches your session intent.

## Lite

- Fast and focused
- Lower visual complexity
- Great for short sessions or quick checks

## Pro

- Multi-panel workflow depth
- Better for active execution and timing
- Useful when managing multiple positions or symbols

# Order entry basics

Keep execution decisions explicit and repeatable.

## Order types

- **Market**: prioritize immediate fill
- **Limit**: prioritize price control
- **Stop**: trigger-based execution for risk and breakout workflows

## Before submit

- Confirm side and size
- Confirm leverage and liquidation implications
- Check spread and top-of-book depth
- Match order type to urgency

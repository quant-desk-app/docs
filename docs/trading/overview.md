# Trading overview

QuantDesk V2 is built on a **unified zero-copy portfolio** architecture: sub-millisecond **account parsing** for bots and the UI, atomic on-chain state updates, and Redis-backed order-book cache reads where enabled. End-to-end click-to-fill latency still includes wallet signing, RPC, and network time.

QuantDesk offers two terminal modes:

- **Lite** for fast monitoring and straightforward actions
- **Pro** for deeper execution context and active workflows

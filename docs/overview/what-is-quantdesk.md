# What is QuantDesk?

QuantDesk is a **perpetual trading terminal and platform** for crypto: built to feel like a serious workstation for execution, context, and risk — not a simplified retail swap with a chart bolted on.

## The gap we care about

Crypto finance still splits the world in awkward ways:

- **Execution** is often split from **context** (depth, flow, position state) and **trust** (custody, fairness, data quality).
- **Pro workflows** (order book, sizing, keyboard speed) are uneven across DeFi; many UIs optimize for the first click, not the hundredth.
- **Builders** need the same rigor: clear APIs, honest limits, and docs that do not pretend the protocol is simpler than it is.

QuantDesk exists to **pull trading surface, data, and integration into one coherent terminal-class experience** — with non-custodial principles and explicit honesty about what is live vs still forming.

## What we’re building

- A **Pro-grade terminal** (and a **Lite** path) for perps: order book, order entry, positions, and workflows that match how active traders actually work.
- A **gateway and APIs** that treat developers as first-class: narrative docs, OpenAPI/Swagger truth, and room to integrate safely.
- A **trust story** that is legible: wallet-first, clear security posture, no hand-wavy “DeFi magic.”
- **Directional** work in **social and collaboration** (strategies, shared context, community tools) — labeled honestly when something is still prototype or roadmap, not “shipped” marketing.

## Who this is for

- **Traders** who want depth without hiding behind complexity.
- **Developers** who need to integrate, automate, or build on the same stack traders use.
- **Anyone** deciding whether the product matches their risk and workflow — we’d rather lose a user on clarity than win one on hype.

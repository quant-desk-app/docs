# Uniques and roadmap

This page intentionally separates what is live now from what is planned. For a full **capability list**, see [Features and tools](./features-and-tools).

## Current strengths

- Pro terminal workflow depth (including registry-driven market browser)
- Lite mode for fast sessions + dedicated backtesting tab
- Backtest → deploy gate → flex publish loop on devnet
- Unified synthetic perps (crypto + US500/US100 index pilots) on V2
- Three-service data plane (Market Data Ingestion Pipeline, Sentiment Feed Aggregator, Social Stream Ingestion Matrix)
- Public docs that prioritize clarity over hype
- Product story that names the **gap in crypto finance** we target — see [What is QuantDesk?](./what-is-quantdesk)

## Product direction

- Mainnet launch hardening and deploy-gate tuning
- E-mini continuous oracle feeds for index perps (beyond SPY/QQQ v1 proxies)
- Copy-trading GA (vault UX still prototype)
- Expanded asset classes per listing policy (post-index pilots)

## Reality policy

QuantDesk docs label status clearly:

- **Shipped**: available now
- **Prototype**: partial or experimental
- **Roadmap**: planned, not yet available

# Careers

<Info>
  We are not running formal open listings yet. This page shares how we work and how to get on our radar for future roles.
</Info>

We're building a serious workstation for on-chain trading — execution, market context, and risk in one coherent terminal. If that problem excites you, we want to know you.

## How we work

<CardGroup cols={2}>
  <Card title="Ship honestly" icon="badge-check">
    We label what's shipped, prototype, or roadmap. No hype, no vaporware.
  </Card>
  <Card title="One action at a time" icon="list-check">
    Small, testable increments over big-bang rewrites.
  </Card>
  <Card title="Depth over surface" icon="layer-group">
    We optimize for the hundredth click, not just the first.
  </Card>
  <Card title="Own the outcome" icon="compass">

  </Card>
</CardGroup>

## Areas we care about

- **Protocol & Solana engineering** — Anchor programs, on-chain matching, oracle integration
- **Trading systems** — order entry, order book, risk, low-latency data plane
- **Frontend / terminal UX** — Lite and Pro surfaces that active traders live in
- **Developer experience** — SDK, APIs, and docs that treat builders as first-class

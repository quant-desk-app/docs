# Brand kit

<Info>
  Downloadable asset packs are coming soon. This page covers the essentials for referencing QuantDesk correctly in the meantime.
</Info>

Guidelines for using the QuantDesk name and marks in articles, integrations, and community content.

## Name

- Write it as **QuantDesk** — one word, capital Q and D.
- Not "Quant Desk", "Quantdesk", or "QUANTDESK" in running text.

## Usage

<Check>Do use the official name and logo to link back to QuantDesk.</Check>
<Check>Do keep clear space around the logo and preserve its proportions.</Check>
<Warning>Don't recolor, stretch, or add effects to the logo.</Warning>
<Warning>Don't imply partnership or endorsement without written permission.</Warning>

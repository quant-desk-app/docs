# Social overview

QuantDesk social features are documented with explicit status labels so there is no confusion between what is live and what is directional.

## Status model

- **Shipped**: available on current devnet-oriented/in-app stacks
- **Prototype**: partially implemented or experimental
- **Roadmap**: planned, not yet available

## Direction

The long-term direction includes:

- Collaboration around strategy workflows
- Shared context for trade reasoning
- Better coordination surfaces for teams and communities

As these capabilities move from roadmap to shipped, this section will be updated with concrete user flows.

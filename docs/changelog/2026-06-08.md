# Week of June 8, 2026

## Updates

- **Faster Lite and Pro terminals.** The Lite dashboard and Pro terminal are now split into smaller, on-demand modules. You’ll see quicker initial loads and snappier tab and window switching, with no visual changes. See [Lite vs Pro](/docs/trading/lite-vs-pro).
- **Slimmer app footprint.** Removed several deprecated trading interfaces, chart prototypes, and legacy order panels that were no longer reachable, trimming the bundle you download in the browser.

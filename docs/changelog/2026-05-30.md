# Week of May 30, 2026

## New features

- **Three-path onboarding with Privy.** Sign up with a wallet, email, or Telegram MPC from a single high-fidelity auth modal that’s now standard across the app. See [Start trading in 5 minutes](/docs/getting-started/start-trading-in-5-minutes).
- **Strategy marketplace.** Publish and discover strategies directly inside QuantDesk, with on-chain performance verification. See [Strategy collaboration](/docs/social/strategy-collaboration).
- **AI agent policy controls.** A new policy widget lets you scope what your agent can do and review actions before they run.
- **Global HUD and collaboration canvas.** New Pro terminal surfaces for shared pulse views and multi-user strategy work.
- **Referral program.** Earn rewards for inviting traders; track activity from the referral dashboard.
- **V2 protocol handlers on devnet.** New on-chain handlers for trading, portfolio, and admin flows, ready to test on devnet.

## Updates

- **Unified wallet button.** A single, themed wallet button now appears everywhere — header, Pro terminal, and settings — backed by Privy so sessions stay consistent across paths.
- **Order book and orders panel refresh.** Cleaner layout, more depth detail, and better state handling for working and historical orders. See [Order book](/docs/trading/order-book) and [Order entry](/docs/trading/order-entry).
- **Deposit modal improvements.** Tighter flow with clearer token selection and sponsorship cues.
- **AI assistant chat upgrades.** Faster responses, better tool-use transparency via a new thought-chain view, and richer message components.
- **Faster RPC routing.** Backend RPC load balancer reduces latency and improves resilience to upstream slowdowns.
- **Hardened protocol security.** Additional invariants and error coverage across V2 trading and collateral flows. See [Protocol and fairness](/docs/security/protocol-and-fairness).

## Bug fixes

- Restored the global header and layout after a navigation regression.
- Fixed high-fidelity UI layout glitches on the landing and Pro pages.
- Anchor V2 compatibility bridge synchronizes auth state correctly for Privy wallets, eliminating stale-session edge cases.
- Resolved several frontend regressions tied to the wallet button migration.

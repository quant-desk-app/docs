# Week of June 29, 2026

A quiet week focused on slimming the backend and stabilizing releases. No user-visible features changed.

## Updates

- **Smaller, faster backend deploys.** The trading backend dropped a large set of unused dependencies and legacy deploy configs, which cuts install time and image size. You should see fewer cold-start delays on API and WebSocket endpoints. See [API overview](/docs/developers/api-overview).
- **Steadier release builds.** A round of dependency cleanup across the trading app, admin tools, and Mikey AI removes a class of transient build failures, so new versions reach you on a more predictable cadence.

# Security and trust

QuantDesk uses non-custodial wallet-first flows for user authentication and signing.

## Public trust posture

- Wallet keys remain in user wallets
- Session/auth controls are enforced server-side
- Public docs focus on user trust and safe usage

Detailed internal security runbooks and audits stay repository-only.
